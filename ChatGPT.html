Dalam Laravel, routing digunakan untuk menentukan bagaimana aplikasi merespons permintaan pengguna ke URL tertentu. Berikut adalah penjelasan dasar dan contoh tentang routing di Laravel:

1. Basic Route

Route paling sederhana hanya membutuhkan URL dan closure (fungsi anonim).

php
use Illuminate\Support\Facades\Route;

Route::get('/hello', function () {
    return 'Hello, World!';
});
  • Route::get: Digunakan untuk permintaan HTTP GET.
  • /hello: URL yang dituju.
  • Fungsi closure akan dieksekusi saat URL diakses.

2. Route dengan Controller

Kita bisa menggunakan controller untuk menangani logika yang lebih kompleks.

php
use App\Http\Controllers\HomeController;

Route::get('/home', [HomeController::class, 'index']);
  • [HomeController::class, 'index']: Memanggil metode index pada HomeController.

3. Route dengan Parameter

Kita bisa menerima parameter dari URL.

php
Route::get('/user/{id}', function ($id) {
    return "User ID: " . $id;
});
  • {id}: Parameter dinamis.
  • $id: Akan otomatis diisi dengan nilai parameter.

Parameter opsional:

php
Route::get('/user/{name?}', function ($name = 'Guest') {
    return "Hello, " . $name;
});

4. Route Grup

Untuk pengelompokan dengan middleware atau prefix.

php
Route::prefix('admin')->group(function () {
    Route::get('/dashboard', function () {
        return 'Admin Dashboard';
    });
    Route::get('/settings', function () {
        return 'Admin Settings';
    });
});

Semua URL dalam grup ini akan diawali dengan admin/.

5. Route Resource

Untuk aplikasi RESTful, gunakan resource routing.

php
use App\Http\Controllers\PostController;

Route::resource('posts', PostController::class);
  • Ini otomatis membuat route seperti:
    • GET /posts index
    • GET /posts/create create
    • POST /posts store
    • dan lainnya.

6. Named Routes

Memberikan nama pada route agar mudah digunakan.

php
Route::get('/profile', function () {
    return 'User Profile';
})->name('profile');

// Menggunakan nama route
$url = route('profile');

7. Middleware pada Route

Menambahkan middleware ke route tertentu.

php
Route::get('/dashboard', function () {
    return 'Dashboard';
})->middleware('auth');

Jika ada kasus spesifik yang ingin kamu buat dalam route, beri tahu, saya bisa bantu langsung!
